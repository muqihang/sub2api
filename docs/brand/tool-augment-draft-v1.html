关于逐梦 Augment Code

逐梦 Augment Code 和官方版的关系是什么？

逐梦保留 Augment Code 的核心上下文引擎与 Agent 能力，将底层的模型推理层接入我们的统一网关。本质上你用的是原版的客户端体验，配上更具性价比的模型路由。

DeepSeek 真的能替代 Claude / GPT 吗？

日常补全、代码理解、单文件改造这类任务，DeepSeek-V3 已与顶级模型几乎持平，且成本不到 1/40。复杂的跨服务重构 / Agent 长任务，我们的智能路由会自动切换到 Claude Sonnet 4.6 或 GPT-5。你拿到的始终是最匹配场景的模型。

我的代码会不会被你们或者模型厂商存下来？

逐梦不存储用户对话与代码内容，仅记录 token 用量用于计费。模型厂商侧的数据策略详见服务条款。企业版客户可选私有部署，所有路由在内部网络完成。
