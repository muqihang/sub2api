常见问题

你想问的，我们都答了。

逐梦的工具和原版（Augment / Claude Code）有什么关系？

我们尊重原版工具，并为用户在中国 / 亚太地区使用做了网络与模型路由的深度优化。所有桌面工具均保留原版核心体验，注入的是模型路由层，让用户可以在 Claude / DeepSeek / GPT 之间灵活选择最具性价比的方案。

我的代码 / 数据会被上传到你们服务器吗？

逐梦只作为模型路由网关存在，不存储你的对话内容与代码。企业版提供私有化部署选项，让模型路由与数据完全在你的内部网络中运行。

团队版和企业版的核心区别是什么？

团队版是公有云 SaaS 模式，开箱即用、统一计费、用量看板。企业版在此基础上增加私有化部署、定制 Agent 开发、SSO / 审计 / 合规对接、SLA 保障与专属客户经理。

可以试用吗？

个人版注册即送 100 万 token 免费体验额度。团队版提供 14 天 5 人免费试用，企业版可申请定制 POC。
