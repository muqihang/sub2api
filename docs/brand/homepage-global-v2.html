FAQ

常见问题Common questions

Workstation 跟 LiteLLM / OpenRouter 有什么不一样?How is Workstation different from LiteLLM / OpenRouter?+

LiteLLM 是 Python 库,要写代码起 Docker;OpenRouter 是云服务,key 经过他们的服务器。Workstation 是桌面 App + 本地网关,装一次就有 GUI 配置;而且我们做了协议适配层,让 DeepSeek 这种非 GPT 模型能在 Codex / Claude Code 等工具里以近乎原生体验运行 —— 这一层是这两家没有的。 LiteLLM is a Python library — code or Docker required. OpenRouter is hosted; your keys traverse their servers. Workstation is a desktop app + local gateway — install once, GUI to configure, keys stay local. And our protocol adapter layer makes non-GPT models like DeepSeek run with near-native feel inside Codex / Claude Code — something neither does.

Codex / Claude Code 是不是要我自己装?Do I need to install Codex / Claude Code myself?+

是的。Workstation 引导你从官方渠道安装原版工具(npm / pip / 官方安装包),我们只负责适配和配置。这样合规干净,而且原版的更新你也能第一时间用到。 Yes. Workstation guides you to install the original tools from their official channels (npm, pip, official packages). We handle adaptation and configuration only. This keeps things clean and lets you get upstream updates immediately.

我的 API key 安全吗?Are my API keys safe?+

Key 加密存储在你本地的桌面 App 里,不上传不经过我们的服务器。Workstation 的本地网关代码我们正在分阶段开源,你可以审计每一个字节怎么处理你的 key。 Keys are encrypted locally, never uploaded, never proxied through our servers. Workstation's local gateway code is being progressively open-sourced — audit every byte that touches your key.

Cloud Bridge 跟 OpenRouter 有什么区别?How does Cloud Bridge compare to OpenRouter?+

Cloud Bridge 由我们的姐妹品牌 RoutePort 提供,功能类似 OpenRouter,但有两点不同:(1) 在 Workstation 里一键开关,跟本地路由共存;(2) 仅向境外用户提供。 Cloud Bridge is powered by our sister brand RoutePort. Similar to OpenRouter in spirit, with two differences: (1) one-click toggle inside Workstation, coexisting with local routing; (2) available outside Mainland China only.

没有 VPN 能装 Claude Code / Codex 这些工具吗?Can I install Codex / Claude Code without a VPN?+

可以。Workstation 自动检测网络环境,国内网络下自动用 npmmirror 等公共镜像源安装(秒级速度),装完恢复你原本的 npm 配置。 Yes. Workstation auto-detects your network. In Mainland-only environments, it temporarily switches npm to public mirrors (like npmmirror) for fast installs, and restores your original npm config after.

你们跟 Augment / Anthropic / OpenAI 是合作关系吗?Are you affiliated with Augment / Anthropic / OpenAI?+

不,Workstation 是独立第三方工具,提供与上述厂商工具的兼容能力和增强体验。Codex、Augment、Claude 等商标分别为各自所有者所有,本产品与 OpenAI、Anthropic、Augment 公司无关联。 No. Workstation is an independent third-party tool that provides compatibility with and enhancements for those products.
