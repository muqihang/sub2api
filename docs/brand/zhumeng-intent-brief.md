# 逐梦品牌与产品定位意图 Brief

本文档用于沉淀我们最新对齐的产品与商业意图，作为后续品牌设计、首页改版、工具页规划、文档入口规划的**核心基准**。

这不是最终文案，也不是视觉设计稿。后续应由具备审美与品牌表达能力的设计团队（或前端开发团队）继续基于本文进行品牌语言、信息架构、页面叙事和视觉方案设计。

---

## 1. 核心症结与战略跃迁：从“卖电”到“卖电器”

**现状与困境：**
Sub2API 原本是一个 API 中转站，提供各家模型的调用。但单纯提供 API（卖 token）等同于“卖电”，用户面临“买度数但不知道接什么电器”的窘境。不仅消耗低，而且极易陷入同质化的“低价内卷”红海。

**战略升级 (PLG 模式)：**
我们不能只是提供 API 卖 token。我们要转型为提供具体的高价值工具和落地方案（卖电器）。
用户为了用上好用的“电器”（如改造后的 Augment、DeepRotor），自然会连上我们的“电网”（消耗 token）。在这个过程中，用户买的是“工作效率”、“学习能力”和“企业落地方案”，从而跳出底层的价格战，进入提供解决方案的蓝海市场。

---

## 2. 品牌定位与价值观重塑

我们不再是一个“披着工具外衣的 API 中转站”，我们要从骨子里成为一个 **AI 时代的先锋工具站与开发者 Hub**。

*   **品牌定位：** 前沿 AI 工具发现与增强平台 (Discover & Supercharge AI Tools)
*   **核心口号（Slogan 建议）：**
    *   *面向C端：* 构建你的下一代 AI 工作流。 (Build your next-gen AI workflow.)
    *   *面向B端：* 让全球最顶尖的 AI 生产力，低成本平滑落地。
*   **核心价值观（品牌基因）：**
    1.  **极效与普惠 (Efficiency & Inclusivity)：** 将消耗极高的顶级工具，通过模型注入（如 DeepSeek）变成人人用得起的平民级利器。
    2.  **开箱即用 (Plug & Play)：** 消灭配置焦虑。复杂的网络和环境已由平台魔改配置好，下载即用。
    3.  **场景驱动 (Scenario-Driven)：** 不谈空泛的大模型，只谈具体场景——写代码、读论文、团队协作。

## 3. 首页设计架构 (The Hub)

首页的任务是**建立权威感、展示工具矩阵、引导分流**。它应该像一个极具未来感的“应用商店”或“极客实验室”，而不是一个充值卖卡的后台。

**第一屏：Hero 区（定调与愿景）**
*   **大标题：** 释放顶级 AI 工具的真正潜力。
*   **副标题：** 精选全球前沿 AI 工具，深度优化与模型注入。无论是极客开发者、科研人员还是企业团队，在这里找到最契合你的 AI 工作流，并且——**成本骤降 80%**。
*   **行动呼唤 (CTA)：** `探索工具库` / `查看企业方案`

**第二屏：核心应用矩阵 (The "Appliances")**
用高质感卡片展示工具，悬浮时展示动画或核心亮点，点击进入专属单页。
*   **逐梦 Augment Code：** 极速上下文引擎，AI 结对编程。
*   **DeepRotor 学术版：** 论文阅读、深度研究与团队学习。
*   **CodeX / Claude Code 桌面版：** 全局注入，多模型混用，超低成本。

**第三屏：我们的“魔法”在哪里？（核心优势）**
*   **打破模型壁垒：** 突破原版工具只能用高昂官方模型的限制，完美注入 DeepSeek 等高性价比模型。
*   **企业级网络与调度：** 告别网络波动与并发限制，企业级专属线路，稳定如一。
*   **统一账户与计费：** 一个逐梦账号，通行所有顶尖 AI 工具，支持团队统一下发额度、用量审计。

**第四屏：企业与组织落地方案 (B2B 钩子)**
*   面向研发团队、高校科研、企业大客户的专属控制台与协同方案展示。引导获取演示或联系专属客服。

**第五屏：底层引擎（面向老手）**
*   向纯极客群体说明：我们依然提供兼容 OpenAI 格式的极致纯粹的底层 API 接口，一键替换。

## 4. 工具单页设计逻辑 (Landing Pages)

每个工具单页就是一个产品的“推销专场”，逻辑必须是：**痛点放大 -> 核心魅力展示 -> 我们的独特优势 -> 一键开始**。每个页面应像一个独立的产品官网。

### 案例 1：【逐梦 Augment Code】
*   **痛点切入：** 原版强大但绑死特定昂贵模型，国内使用门槛高。
*   **改造与优势：** 独家技术注入 DeepSeek 等低成本高智商模型；原封不动保留其强大的“超长上下文感知引擎”（懂你的整个 codebase）。
*   **视觉：** 丝滑的代码补全、跨文件问答的实际操作动图。
*   **转化路径：** 注册获取逐梦定制版并自动配置 API。

### 案例 2：【DeepRotor 逐梦版】（学习教育/科研协作）
*   **工具价值：** 介绍原开源工具的强大（不仅是翻译，更是学术对话、构建个人知识图谱）。
*   **改造与优势：** 开箱即用（免部署）；模型自由（精读与泛读模型无缝切换）；支持云端同步与导师/团队统一发卡协作。

### 案例 3：【逐梦 CodeX / Claude Code】
*   **改造与优势：** 打破官方模型壁垒，提供灵活的模型选择策略，在保留原版丝滑体验的同时，通过替换模型引擎大幅降低日常使用费用。

---

## 5. 文档与开发者控制台 (Workspace)

*   **文档中心：** 文档不仅是说明书，更是信任感来源。包含各工具接入指南、团队协作配置、大客户落地案例等。
*   **开发者控制台：** 摒弃“买卡/充值”的廉价感，升级为用户的 **Workspace（工作区）**。提供数据看板 (Analytics)、团队成员管理、API Keys 下发等企业级管控功能。

---

## 6. 给设计团队的执行建议

1.  **抛弃“代充/代理”的视觉风格：** 整体体验向 Vercel, Stripe, Raycast 等世界级开发者工具靠拢。
2.  **内容驱动设计：** 重点设计工具卡片、案例展示、解决方案板块，让整个平台看起来内容丰富、技术硬核。
3.  **强化“解决方案”体感：** 哪怕背后最终消耗的是我们的 API，前端展示的一定要是“集成方案”和“落地指南”。

> 总结：我们分享全球最前沿的 AI 工具落地经验（面子，负责引流建立权威），并顺带提供一个极其稳定、极具性价比的统一引擎，让这些经验能一键落地（里子，负责完成商业闭环）。
