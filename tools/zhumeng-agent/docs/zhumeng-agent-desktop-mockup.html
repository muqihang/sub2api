为什么首版不上架 Mac App Store
✓需要写入 ~/.codex 配置并注册 zhumeng-agent:// 深链
✓需要运行本机代理 127.0.0.1:<port>
✓可能修补 Codex.app/app.asar 以启用模型选择和插件市场
未签名内测包会被 Gatekeeper 强提示；未公证包可能显示"无法验证"。正式官网发布必须 Developer ID 签名 + Apple Notarization + SHA256 校验。
